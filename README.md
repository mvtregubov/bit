# Интерактивная презентация (PHP 8.2 + MySQL 5.7)

Готовый шаблон презентации: один слайд = один экран, полностью адаптивен. Данные приходят из MySQL через PHP 8.2. В комплекте — SQL дамп с тестовыми слайдами.

## Быстрый старт
1. Скопируйте все файлы в одну папку (например, `/home/c/cd65093/doorhan.company/public_html/subdomains/slides/`).
2. Создайте базу и наполните данными:
   ```bash
   mysql -u root -p < dump.sql
   ```
3. Обновите доступы к БД в `config.php`.
4. Запустите встроенный сервер PHP прямо из этой папки:
   ```bash
   php -S 0.0.0.0:8000
   ```
5. Откройте [http://localhost:8000](http://localhost:8000) и листайте стрелками или свайпами.

## Структура
- `index.php` — одиночная страница слайдов.
- `slides.php` — API, отдаёт слайды из MySQL (fallback на демо-данные в JS).
- `Database.php`, `PresentationService.php` — соединение и выборка слайдов через PDO.
- `dump.sql` — создание базы `presentation_db` и тестовые записи.
- `styles.css`, `app.js` — стили и интерактивная логика.

## Настройки
- PHP 8.2 и PDO MySQL.
- MySQL 5.7 (дамп совместим).
- Все стили и скрипты адаптивны, поэтому презентация смотрится одинаково на десктопе и мобильных устройствах.

# Интерактивная презентация (PHP 8.2 + MySQL 5.7)

Готовый шаблон презентации: один слайд = один экран, полностью адаптивен. Данные приходят из MySQL через PHP 8.2. В комплекте — SQL дамп с тестовыми слайдами.

## Быстрый старт
1. Создайте базу и наполните данными:
   ```bash
   mysql -u root -p < db/dump.sql
   ```
2. Обновите доступы к БД в `config.php`.
3. Запустите встроенный сервер PHP из папки `public`:
   ```bash
   php -S 0.0.0.0:8000 -t public
   ```
4. Откройте [http://localhost:8000](http://localhost:8000) и листайте стрелками или свайпами.

## Структура
- `public/index.php` — одиночная страница слайдов.
- `public/api/slides.php` — API, отдаёт слайды из MySQL (fallback на демо-данные в JS).
- `src/Database.php`, `src/PresentationService.php` — соединение и выборка слайдов через PDO.
- `db/dump.sql` — создание базы `presentation_db` и тестовые записи.
- `public/css`, `public/js` — стили и интерактивная логика.

## Настройки
- PHP 8.2 и PDO MySQL.
- MySQL 5.7 (дамп совместим).
- Все стили и скрипты адаптивны, поэтому презентация смотрится одинаково на десктопе и мобильных устройствах.
